# Day 03 – Learning Database Basics for CRM

## What I Did Today

- Learned what a database is and why it is needed
- Understood how CRM stores data in tables
- Identified basic tables needed for the CRM project

## Tables Planned

- users → store admin and staff login details
- customers → store customer information
- leads → store potential customer details
- sales → store sales records

## Basic Fields (Example)

- users: id, name, email, password, role
- customers: id, name, phone, email, address
- leads: id, customer_id, status
- sales: id, lead_id, amount, date
